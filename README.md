# teaching-log repo

## What is this repository?

This repo is a **teaching log** for this Python/Git course.
Think of it as an **interactive textbook with history**.

This repo is **read-only**. 

You do **NOT** need to do anything.

This is meant for you to look back at 
* what was covered in class
* **how the code evolved** and
* compare code from this repo to your code in your living-code repo
---

## How to use this repository (no Git required)

### 1. Browse files
- Click on any `.py` file to read the code

### 2. View commit history
- Click the **“Commits”** button near the top of the repository
- Read the commit messages — they explain *why* a change was made
